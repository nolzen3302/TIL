집합 자료형
===


## 1. 집합 자료형의 특징
- 집합은 중복을 허용하지 않는다
- 집합은 순서가 없다

## 2. 집합의 형변환
집합을 리스트로 만드는 방법
```python
list_set = set(리스트명)
```
리스트를 집합으로 만드는 방법
```python
list(집합명)
```

## 3. 집합자료형의 연산
### 3.1 합집합
- set1.union(set2)
- set1 | set2
   
### 3.2 교집합
- set1.intersection(set2)
- set1 & set2   
   
### 3.3 차집합
- set1.dfference(set2)
- set1-set2
