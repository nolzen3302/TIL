불 자료형
===
불은 오직 True(참)와 False(거짓)값만 가질 수 있다.

## 1. 연산자
### 1.1 비교 연산자
| 연산자 | 의미 |
| :--: | :--: |
| \== | 같다 |
| != | 다르다 |
| < | 작다 |
| \> | 크다 |
| <= | 작거나 같다 |
| \>= | 크거나 같다 |

### 1.2 논리 연산자
| 연산자 | 의미 |
| :--: | :--: |
| not | 아니다 |
| and ( & ) | 그리고 |
| or ( \| ) | 또는 |

### 1.3 in 연산자
- x in _문자열/리스트/딕셔너리.keys()/딕셔너리.values()..._   
- x not in _문자열/리스트/딕셔너리.keys()/딕셔너리.values()..._

## 2. 기타
- True : 0이 아닌 모든 수, 비어있지 않은 모든 그룹형 변수
- False : 0이거나 비어있는 모든 그룹형 변수

```
int(True) -> 1
float(True) -> 1.0
int(False) -> 0
float(False) -> 0.0
```
